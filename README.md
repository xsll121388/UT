# UT - 一个使用ai制作可以填词的修音小玩具

基于 RMVPE 音高提取 + NSF-HiFiGAN 声码器的音高修正与音频编辑工具。

## 功能特性

- **音高提取**：使用 RMVPE 模型自动提取音频音高（F0 曲线）
- **音高修正**：自动将音高对齐到指定音阶（大调、小调、五声音阶等）
- **手动绘制**：支持手动绘制/编辑音高曲线
- **MIDI 音符编辑**：移动、缩放、切割音符，支持网格吸附
- **拉伸控制点**：在音符内部添加控制点，精细调整时序
- **实时预览**：修改后即时播放效果
- **歌词输入**：批量为音符填写歌词
- **音源文件夹**：通过音源样本重建音频时间线
- **批量处理**：支持批量音频文件处理
- **多种导出格式**：WAV / FLAC / AIFF / OGG

## 环境要求

- **Python 3.10 或更高版本**
- **Git**（含 Git LFS，用于下载模型文件）
- **Windows**（推荐）/ macOS / Linux

## 安装

### 1. 克隆仓库

```$env:GIT_LFS_SKIP_SMUDGE = "1"
git clone https://github.com/xsll121388/UT.git
cd UT
git lfs pull
```
> **注意**：本项目使用 Git LFS 管理模型文件（约 400MB）。确保已安装 [Git LFS](https://git-lfs.com/)，否则模型文件可能无法正常下载。

### 2. 安装依赖

**方式 A：使用安装脚本（推荐）**

```bash
install_deps.bat
```

**方式 B：手动安装**

```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 3. 运行程序

```bash
python main.py
```

或双击 `start.bat` 启动。

## 快速开始

### 第一步：导入音频

1. 点击菜单栏 `文件 → 打开音频`，或直接拖拽音频文件到窗口
2. 支持 WAV / FLAC / MP3 / OGG / AIFF 格式
3. 程序会自动提取音高（F0 曲线）并开始渲染

### 第二步：导入 MIDI（可选）

1. 点击 `文件 → 导入 MIDI`
2. 导入后音频会自动对齐到 MIDI 音符的时长和音高

### 第三步：导入音源文件夹（可选）

1. 点击 `文件 → 导入音源文件夹`
2. 选择包含按歌词命名音频文件的文件夹（如 `啊.wav`、`哦.flac`）
3. 程序会根据 MIDI 音符的歌词自动组装音频时间线

### 第四步：编辑音高

程序提供三种编辑模式：

| 模式 | 快捷键 | 功能 |
|------|--------|------|
| **绘制音高** | F1 | 用鼠标绘制/修改音高曲线 |
| **音符编辑** | F2 | 移动、缩放、切割 MIDI 音符 |
| **切割** | F3 | 快速切割音符 |

### 第五步：调整参数

右侧参数面板可调整：

- **调性**（C、D、E 等）
- **音阶类型**（大调、小调、五声音阶等）
- **修正速度**（Retune Speed）：控制自动修正的程度

### 第六步：输入歌词

在底部歌词栏输入音节（空格分隔），选中音符后点击 **Apply** 即可批量赋值。

### 第七步：导出

点击 `文件 → 导出音频`，选择格式和保存路径。

## 高级功能

### 拉伸控制点

在音符编辑模式（F2）下，右键点击音符可：
- **添加拉伸控制点**：在音符内部添加可变边界
- **拖动控制点**：调整音符内部的时序比例
- **删除/清除控制点**：恢复原始时序

### 自动对齐 MIDI 音高

点击工具栏的 `🎵 对齐 MIDI` 按钮（或按 F3），音频音高将自动拉平到 MIDI 音符。

### 批量处理

点击 `文件 → 批量处理`，可对多个音频文件应用相同的参数设置。

## 工程文件

- 保存格式：`.ut`
- 包含：音频路径、MIDI 音符、音高数据、参数设置
- 可通过 `文件 → 保存工程` / `打开工程` 管理
